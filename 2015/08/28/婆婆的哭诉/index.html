婆婆的哭诉

发表于 2015-08-28   |   分类于 思悟   |  

前言

前两天去婆婆家，向我哭诉了很多内心的想法。

她说的话也许是很多老年人的心声，所以我想聊聊这个事，希望正在看文章的您能静下来，抽一点点时间反思。

对话

暑假回来了快一个月，一直没去见婆婆，理由是：忙于找工作。趁着网易还没有出结果，我可以暂时休息一下，就打算抽点时间去拜访婆婆。

进门的时候婆婆说“贤贤来了啊，我以为你记不到婆婆了，不来看婆婆了！”，声音带着轻微的哽咽。

自从爷爷去世后，婆婆大多数时候是一个人生活，偶尔会去子女家住住。

她像往常一样，去内屋拿糖给我吃。

“前段时间一直在准备校招，太忙了，昨天才从杭州回来。你看我这不是一闲下来就过来看你了吗？”，我解释到。

“没事，婆婆晓得你们都忙。我一个人也过得很快乐，很坚强。”，声音微微颤抖着。

“到时候如果能进网易，年薪十几万，我会好好孝敬婆婆的！”。

“婆婆不缺钱。只要你们过得好，过得比我强，婆婆就很开心！”。

…

“婆婆不想去打扰你们，婆婆觉得自己很无能！”，婆婆情绪突然激动起来。

“婆婆，你为啥这样说呢！？”，我惊愕道。

“我走到哪里都被挤兑。你晓得你外婆性格强，她挤兑我，我只有走远点，不敢来你们家。但是你外婆确实能干，能把你们家维护好，我想也就算了，只要你们好就好！到幺爸家呢…到姑妈家呢…”，婆婆含着眼泪说。

“小孙孙又不懂事，你和张沁又回来得少。有时候你爸爸和姑妈带我去耍，结果自己在打麻将，我只有一边去耍…”，婆婆眼泪止不住流了下来。

我陷入了深深的自责，正想安慰婆婆两句，她打断了我“没事，婆婆很坚强！”，她一边用褶皱的手擦着眼泪，一边从包里掏出500块钱，塞到我手里，让我拿去用，“平时都是外婆在带你，婆婆只有用钱表示一哈了！”…

思考

1. 我真的抽不出时间吗？
抽不出时间我会有空去旅行？抽不出时间我会有空去参加聚会、去看电影？这明显是借口。

2. 陪是真心真意地陪吗？
我们通常会在石刻公园、东湖山组织家庭聚会，或者在四川周边自驾游。自驾游还行，一般都是逛风景，会带着老年人一起逛。但是在东湖山、时刻玩，通常就是父辈们打麻将，我在一边看书、玩手机，然后老年人在一边自己玩，而这种情况占了多数。看似是在陪，然而我觉得这种陪并不是他们的心理诉求。

我印象中，他们真正开心、幸福的时候，是在和子孙说话的时候，强调一点，是在子孙积极回应而不是随意敷衍的时候。

总结

由于子孙的挤兑与不陪伴，婆婆“觉得自己很无能”，最后只能用钱来表明自己还有一些用。我真的非常心寒，非常心痛！

婆婆是抚养过我的亲人。她现在75岁，所剩时日并不多了。

我希望能做一些实际的行动，填补婆婆受伤的心灵。我告诉婆婆以后不管在哪里，每周都会打一个电话给她，一定要做到！

后记

我妈给我上了一课。她提到了一点我没有想到的：婆婆不愿意磨合。

比如说我姑妈家养了一只狗，婆婆不喜欢狗，有时候会骂它甚至打两下，于是母女之间会产生一些分歧，而婆婆不愿意磨合。随着类似事情的积累，婆婆会觉得处处都在挤兑她。说到底，也许是婆婆的调节能力还不够强大、胸怀还不够宽广。

老妈问我这个事要怎样处理呢？如何让她快乐、幸福地度过晚年生活呢？

就我的经验来看，想改变老年人太困难了，所以只有自己做些改变。比如说以后陪她的时候，多聊聊天、散散步，少玩手机、少打牌；每周打个电话。

我妈说以后子女每周都带她出去玩一趟。
